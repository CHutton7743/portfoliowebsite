Project Character Forge

DnD Character Generator

Technologies this project uses: SPRING, JPA, REACT.js, AWS Cloud RDS, AWS EC2, Java, Javascript

Creating a character for dungeons and dragons can be a daunting and overwhelming task for newer players.. At least that has been my experience with bringing new players into my games. So in order to resolve this, I decided I would create a website to help them out. The end goal for this website is for players to be able to log in to their account, and be able to create, save, update and delete characters that they have made. They will start off simple, entering player name, race, class. From there the program will lead them through designing their character in an intuitive and easy way that should take minutes instead of hours of research.

Currently the back end spring program for this project is working and supports REST methods for CRUD operations for creation of characters, having never worked with REACT.JS and only light javascript knowledge in general, I expect the react front end to take the longest part for this project. Currently I am working on getting it set up for basic operations using libraries such as material ui. From there I will be working on getting the front end to communicate with the back end. The final steps for this project will be to implement an amazon RDS for the application, and an EC2 instance to get it up and runnning.
